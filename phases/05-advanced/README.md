# Phase 5: Advanced Topics & Open Source

**Duration:** 6-8 weeks
**Prerequisites:** Phase 4 (algorithms, hardware experience)
**Capstone:** qec-toolkit — QEC simulation and visualization toolkit + merged open-source PR

## Objectives

By the end of this phase, you will:
- Implement quantum error correction codes
- Understand the stabilizer formalism and surface codes
- Evaluate quantum ML claims critically
- Understand topological approaches to quantum computing
- Contribute to a major open-source quantum project

## Weekly Plan

### Weeks 1-3: Quantum Error Correction
- Classical error correction review (repetition, Hamming)
- Bit-flip and phase-flip codes
- Shor's 9-qubit code
- Steane code and CSS codes
- Surface codes (toric code)
- Stabilizer formalism
- Threshold theorem and fault tolerance
- Exercises: implement each code, simulate error correction

### Weeks 4-5: Quantum Machine Learning & Topological QC
- Quantum kernels and feature maps
- Variational quantum classifiers
- Barren plateaus and trainability
- Critical evaluation of quantum advantage claims
- Anyons, braiding, and topological codes
- Exercises: implement quantum classifier, compare to classical

### Weeks 6-8: Open Source Contribution & Capstone
- Deep-dive into Qiskit/Cirq/PennyLane source code
- Find and claim a good first issue
- Implement, test, submit PR
- Build qec-toolkit: error correction simulation + visualization

## Resources

- *Quantum Error Correction* — Lidar & Brun
- *Quantum Computation and Quantum Information* — Nielsen & Chuang (Ch. 10)
- PennyLane QML demos
- Qiskit/Cirq/PennyLane contributing guides
