# Phase 4: Quantum Algorithms

**Duration:** 6-8 weeks
**Prerequisites:** Phase 3 (circuits, gates, Qiskit)
**Capstone:** algorithm-suite — Complete algorithm implementations + IBM hardware runs

## Objectives

By the end of this phase, you will:
- Implement all major quantum algorithms from scratch
- Understand the quantum speedup each algorithm provides
- Run algorithms on IBM Quantum hardware
- Apply error mitigation techniques to noisy results
- Implement variational algorithms for NISQ-era problems

## Weekly Plan

### Weeks 1-2: Foundational Algorithms
- Deutsch and Deutsch-Jozsa algorithms
- Bernstein-Vazirani algorithm
- Simon's algorithm
- Exercises: implement each, prove speedup, run on simulator

### Weeks 3-4: QFT & Major Algorithms
- Quantum Fourier Transform (build up from DFT)
- Quantum phase estimation
- Shor's factoring algorithm
- Grover's search algorithm
- Exercises: factor small numbers, search unstructured databases

### Weeks 5-6: Variational Algorithms (NISQ)
- Variational Quantum Eigensolver (VQE)
- QAOA for combinatorial optimization
- Parameterized circuits and gradient computation
- Classical-quantum hybrid optimization loops
- Exercises: find ground state of H2, solve MaxCut

### Weeks 7-8: Hardware & Capstone
- IBM Quantum account setup
- Transpilation and hardware constraints
- Error mitigation techniques
- Run algorithms on real hardware, compare to simulator
- Build algorithm-suite with all implementations

## Resources

- *Quantum Computing: An Applied Approach* — Hidary (Ch. 8-14)
